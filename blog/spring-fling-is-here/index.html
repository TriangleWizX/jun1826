What EBI-Style Overtime Means

Our event uses an EBI-style submission-only format: no points in regulation, then controlled overtime from defined finishing positions if needed.

Regulation is commonly described as 10 minutes, submission only, then overtime starting from back control or spiderweb with alternating attack and escape turns; if neither athlete gets a submission, escape time can decide the winner. FloGrappling

Many EBI-format events run a single overtime round in bracket matches and up to 3 overtime rounds in finals. Submission Challenge See Core Culture pricing.
