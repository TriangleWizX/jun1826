# Weekly KPI Dashboard (Booked Intros + Local SEO)

## Primary Metrics
- `book_intro_submit` count (weekly)
- Booking conversion rate (`book_intro_submit` / sessions on booking pages)
- Contact conversion rate (`contact_form_submit` / sessions on contact page)

## Supporting Metrics
- `book_intro_click` by `page_type` and `cta_position`
- `sms_click` and `phone_click` by `page_type`
- Top landing pages -> intro submit path
- Top UTM campaign/source/medium -> intro submit

## Local SEO Hygiene Checks
- Indexed duplicates count (`/contact.html`, `/contact/`, legacy slugs)
- Branded query SERP title/snippet consistency
- Core page structured-data consistency

## Weekly Review Cadence
- Monday: export prior week GA4 event counts
- Tuesday: compare week-over-week conversion deltas
- Wednesday: select one CTA copy/layout experiment
- Friday: publish one local-intent content update
